# 🌌 NebulaX

NebulaX is a full-stack space exploration web application that combines real-time NASA data, ISS tracking, AI-style summaries, and user authentication.

🔗 Live Website: https://nebula-x-eight.vercel.app/

---

## 🚀 Features

- 🌍 Interactive space-themed UI with video background
- 🛰 Real-time ISS location tracking
- 🌠 NASA Astronomy Picture of the Day (APOD) integration
- 🧠 AI-style compressed cosmic insights
- 🔐 JWT-based authentication (Register / Login)
- ⭐ User-specific Favorites system
- ☁️ Deployed full-stack architecture

---

## 🛠 Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Deployed on Vercel

### Backend
- Node.js
- Express.js
- JWT Authentication
- PostgreSQL (Supabase)
- Deployed on Render

### APIs Used
- NASA APOD API
- Open Notify ISS API

---

## 🧠 Architecture

Frontend → Vercel  
Backend → Render  
Database → Supabase PostgreSQL  

Secure environment variables with .env  
Protected API routes using JWT
